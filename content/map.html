std::map In-Depth

An associative container that stores key-value pairs in sorted order. Implemented typically as a balanced binary search tree (often Red-Black tree). Header: #include <map>

Key Performance Characteristics: O(log n) search, insertion, and deletion. Keys are automatically sorted. Unique keys only.

Basic Operations

Construction and Insertion
std::map<std::string, int> ages;

// Insert using subscript operator
ages["Alice"] = 30;
ages["Bob"] = 25;

// Insert using insert()
ages.insert({"Charlie", 35});
ages.insert(std::make_pair("David", 28));

// Insert using emplace
ages.emplace("Eve", 32);

// Result: sorted by key alphabetically
// Alice: 30, Bob: 25, Charlie: 35, David: 28, Eve: 32
Finding and Accessing Elements
auto it = ages.find("Alice");
if (it != ages.end()) {
    std::cout << "Alice is " << it->second << " years old" << std::endl;
}

// Using at() - throws exception if key doesn't exist
try {
    int age = ages.at("Frank"); // throws std::out_of_range
} catch (const std::out_of_range& e) {
    std::cout << "Frank not found" << std::endl;
}

// Using operator[] - creates element if it doesn't exist
int default_age = ages["Frank"]; // Creates Frank with value 0

Iteration

Range-based For Loop
// Iterate through all key-value pairs (sorted by key)
for (const auto& [name, age] : ages) {
    std::cout << name << ": " << age << std::endl;
}

// Pre-C++17 syntax
for (const auto& pair : ages) {
    std::cout << pair.first << ": " << pair.second << std::endl;
}

Common std::map Patterns

1. Frequency Counter

Count occurrences of elements using a map.

std::vector<std::string> words = {"apple", "banana", "apple", "cherry", "banana", "apple"};
std::map<std::string, int> frequency;

for (const auto& word : words) {
    frequency[word]++; // Auto-creates entry with 0 if doesn't exist
}

// Result: apple: 3, banana: 2, cherry: 1
2. Custom Comparator

Use custom sorting criteria for keys.

// Sort strings by length instead of alphabetically
std::map<std::string, int, std::function<bool(const std::string&, const std::string&)>>
    length_map([](const std::string& a, const std::string& b) {
        if (a.length() != b.length()) return a.length() < b.length();
        return a < b; // Same length: alphabetical order
    });

length_map["cat"] = 1;
length_map["elephant"] = 2;
length_map["dog"] = 3;
// Order: cat, dog, elephant
